### ClassIsland

ClassIsland 是一款适用于班级多媒体屏幕的课表信息显示工具，可以在 Windows 屏幕上显示各种信息。<br/>
本应用的名字灵感源于 iOS 灵动岛（Dynamic Island）功能。
